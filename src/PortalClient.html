Portal Client is a small utility used in client target machines to register them to the marketplace. Portal Client can manage the requested applications by the marketplace to the target machine.

Starting Portal Client

Use the following command to start the client manually:

java -jar '-Dmarketplace_api_endpoint=http://83.212.106.218:8080/portal/services/api/repo' '-Dportal_client_owner_id=34' '-Dportal_client_owner_username=dev' '-Dportal_client_host_fqdn=example.com' target/gr.upatras.ece.nam.portal.client-0.0.1-SNAPSHOT.war

Replace portal_client_owner_id with your Marketplace user ID

Replace portal_client_owner_username with your Marketplace username

Replace portal_client_host_fqdn with your server FQDN

After a few minutes of operation, Portal Client will successfully register itself to the marketplace, under your account. You can see it if is registered successfully by visiting your Registered Resources

Using Portal Client

By default portal client listens on port 13001 for local requests. To manage it, open a browser and go to http://localhost:13001
